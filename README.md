#### 安装教程

1.  手动安装：打开chrome://extensions/，把cv_helper_extension.crx拖进去
2.  从应用商店安装：已上传至谷歌浏览器插件应用商店（因为审核的原因，早期更新比较频繁，可能会比Gitee慢一些版本）
